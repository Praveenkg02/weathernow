## What it does

Search any city and get:
- 🌡️ Temperature, feels-like, high/low
- 💧 Humidity · 💨 Wind speed & direction · 👁️ Visibility · 🔵 Pressure
- 📅 5-day forecast with weather icons

## Tech Stack

HTML5 · CSS3 · JavaScript (ES6+) · OpenWeatherMap API · GitHub Pages

## Setup (optional — runs in demo mode without a key)

1. Get a free API key at [openweathermap.org](https://openweathermap.org/api)
2. In `index.html`, replace `YOUR_API_KEY_HERE` with your key
